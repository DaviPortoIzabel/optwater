# Low-Cost Leak Detector

## Overview
This project is a low-cost smart water leak detection system using a YF-S201 flow sensor and an ESP32-C3.

## Goal
Detect continuous low water flow that may indicate leaks in plumbing systems.

## Features (planned)
- Real-time flow monitoring
- Leak detection logic
- LED + buzzer alerts
- Custom PCB
- Compact enclosure
